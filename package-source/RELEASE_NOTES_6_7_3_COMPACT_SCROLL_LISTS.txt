REPORT CARD ENTERPRISE v6.7.3 REUSABLE SCHOOLS EDITION
COMPACT OPERATIONAL LISTS RELEASE

BASELINE
- Built from the user-confirmed working v6.7.2 Reusable Schools Edition.
- Frontend-only upgrade. No database schema or Edge Function change is required.

IMPROVEMENTS
1. Student Directory, Teacher Directory, Principal Directory, Users and Access, Report Cards, and Audit Trail tables now use bounded vertical scrolling.
2. Table headings remain visible while scrolling long lists.
3. Notifications now scroll inside a compact panel instead of making the page excessively tall.
4. New Report Card uses a fixed-height scrollable student list after a class is selected.
5. The student chooser supports the mouse wheel, visible scrollbar, touch scrolling, and keyboard arrow keys.
6. Responsive limits keep list panels compact on desktop, tablets, and mobile screens.
7. Pagination remains outside the scrollable table so navigation controls remain visible.
8. The reusable-school package generator now produces v6.7.3 packages containing the same compact-list improvements.

DEPLOYMENT
- Replace the GitHub Pages frontend files with the v6.7.3 GITHUB_PAGES_FRONTEND contents.
- Preserve the live Supabase Project URL and Publishable key in config.js.
- No SQL execution is required.
- No Edge Function redeployment is required.
- Close existing application tabs and perform a hard refresh after GitHub Pages redeploys.
