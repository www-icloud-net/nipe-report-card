REPORT CARD ENTERPRISE v6.8.1 REUSABLE SCHOOLS EDITION
REPORT WORKFLOW AUTHORITY AND CLASS-LEVEL BULK ACTIONS

BASELINE
- Built from the user-confirmed stable production baseline: v6.8.0 Reusable Schools Edition.
- v6.8.0 remains preserved as the rollback and comparison package.

IMPLEMENTED CHANGES
1. CLASS-TEACHER-ONLY SUBMISSION
- Only the officially assigned class teacher can submit a report for Principal approval.
- Subject teachers can continue entering scores for their assigned subjects.
- Subject teachers cannot submit an individual report or use bulk submission.
- System Administrators and Principals cannot submit reports on behalf of the class teacher.
- Submission still requires every assigned subject and every required assessment score to be complete.

2. PUBLICATION AUTHORITY
- Only the assigned class teacher or System Administrator can publish a Principal-approved report.
- Subject teachers cannot publish, republish, register, or refresh an official report PDF.
- Principals approve or return reports but do not publish them.

3. BULK CLASS WORKFLOW
- Assigned class teachers can submit all eligible reports for one selected class and term.
- Principals can approve all eligible submitted reports for one selected class and term.
- Assigned class teachers and System Administrators can publish all eligible approved reports for one selected class and term.
- Bulk publication automatically attempts to generate and store the latest official PDF for each newly published report.

4. FAILURE ISOLATION AND FEEDBACK
- Each report is validated separately through the same secure workflow function used by individual actions.
- Incomplete reports remain unchanged instead of blocking every other valid report.
- The result summary identifies transitioned reports, incomplete or failed reports, reports already in the target status, and students without report records.
- Every successful report transition retains its normal revision, audit, workflow-event, notification, and publication history.

DATABASE UPGRADE
- New permanent upgrade file: 07_schema.sql.
- Existing v6.8.0 installations run only 07_schema.sql.
- Fresh installations run all nine SQL files in order through 07_schema.sql.

FRONTEND
- New Submit class reports button for class teachers.
- New Approve class reports button for Principals.
- New Publish class reports button for assigned class teachers and System Administrators.
- Bulk actions require one selected class and one selected term.
- A class teacher cannot bulk-submit or bulk-publish a class where the teacher is assigned only as a subject teacher.
