GITHUB PAGES FRONTEND: v6.7.0 FINAL BUILD

1. Edit config.js with the Supabase Project URL and browser-safe Publishable key.
2. Upload the CONTENTS of this folder to the root of the GitHub repository.
3. In GitHub Settings > Pages, choose Deploy from a branch, main, /(root).
4. In Supabase Authentication > URL Configuration, set the Site URL and Redirect URL to the final GitHub Pages URL.
5. The exact browser libraries are already included under assets/vendor. Do not replace them with unpinned CDN URLs.
6. Never place a Secret key, service_role key, database password, NIS_CRON_SECRET, NIS_BACKUP_ENCRYPTION_KEY or email-provider API key in this folder.

See COMPLETE_FRESH_SETUP_SUPABASE_TO_GITHUB.md in the complete package.
